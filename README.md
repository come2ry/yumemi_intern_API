## Pre Installation
OS X:

```sh
brew install pyenv
brew install pipenv
```
1. MAMPをダウンロードしてMySQLのポートを3306に指定
2. yumemi_dbを作成

## Installation

```sh
pipenv install
make init
```

## Usage example

```sh
make run
```

http://0.0.0.0:8000/docs
へアクセス
